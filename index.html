Ng1 Datepicker

Simple datepicker for AngularJS 1+

Angular directive that wraps an input tag and opens a calendar on blur event on the input. Inserts a separator (slash) on keyup and returns a string on the DD/MM/YYYY format.

Working example

Scope: {{ $ctrl.data }}

How to use


1. Add the MomentJS library (the only dependency)


2. Add the script and CSS files


<link rel="stylesheet" href="datepicker/datepicker.css">
<script type="text/javascript" src="datepicker/datepicker.js"></script>

3. Add the module


angular.module('app', ['datepickerModule'])

4. Wrap the input with the directive


<datepicker ng-model="date">
  <input type="text" placeholder="dd/mm/yyyy">
</datepicker>

5. Also works with nested divs and custom classes


<datepicker ng-model="date">
  <div class="input-group">
    <input class="form-control" type="text" placeholder="dd/mm/yyyy">
  </div>
</datepicker>
